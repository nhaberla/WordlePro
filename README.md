[![Coverage badge](https://img.shields.io/endpoint?url=https://raw.githubusercontent.com/nhaberla/WordlePro/python-coverage-comment-action-data/endpoint.json)](https://htmlpreview.github.io/?https://github.com/nhaberla/WordlePro/blob/python-coverage-comment-action-data/htmlcov/index.html)
